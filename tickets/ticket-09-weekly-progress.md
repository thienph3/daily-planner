# Ticket 09 — Feature: Theo dõi tiến độ theo tuần

## Mô tả
Xây dựng dashboard tổng quan hiển thị tiến độ hoàn thành task, thói quen và mood trong tuần.

## Công việc
- [ ] Tạo `shared/widgets/weekly_dashboard.dart` — widget dashboard tổng quan
- [ ] Hiển thị tỷ lệ hoàn thành task mỗi ngày trong tuần (bar chart đơn giản)
- [ ] Hiển thị streak — số ngày liên tiếp hoàn thành tất cả task
- [ ] Tổng hợp mood trong tuần (mood phổ biến nhất)
- [ ] Tổng hợp số bữa ăn đã lên kế hoạch
- [ ] Tổng hợp số nhật ký đã viết
- [ ] Tích hợp vào màn hình chính hoặc tab riêng
- [ ] Animation cho progress bars và charts

## Tiêu chí hoàn thành
- Dashboard hiển thị đúng dữ liệu tổng hợp từ các feature
- Cập nhật realtime khi dữ liệu thay đổi
- UI trực quan, dễ đọc
